# 4. zellij.kdl - Terminal multiplexer theme
# 5. btop.theme - System monitor theme
# 6. neovim.lua - Editor theme
# 7. vscode.sh - VS Code theme installation
# 8. background.jpg - Wallpaper (JPEG or PNG, recommended 4K resolution)
